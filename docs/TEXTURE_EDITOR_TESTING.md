# Texture Editor Manual Test Checklist

1. Start a dev client/server.
2. Place a CD Writer.
3. Insert a blank Blueprint CD.
4. Select an audio file.
5. Open the advanced texture editor.
6. Draw a red/blue diagonal pattern.
7. Click Finish and import.
8. Confirm the CD Writer preview changes.
9. Click Print.
10. Confirm the output slot item visually shows the diagonal pattern.
11. Pick up the item.
12. Confirm the inventory item visually shows the diagonal pattern.
13. Hold the item in hand.
14. Confirm the hand item visually shows the diagonal pattern.
15. Drop the item.
16. Confirm the dropped item shows the pattern, or record the fallback seen in-game.
17. Insert it into a jukebox.
18. Confirm playback still works.
19. Restart the world.
20. Confirm the item design persists.
